# Assignment 3

## Key Bindings

- 3,4,5,6 Selecting meshes

  - Default 2 -> none selected
- m Mesh Transformation Mode

  - ArrowKeys -> X, Y translation
  - Middle Mouse button for scaling(+/-)
- s Shading model choice
  - continuous pressing to set the mode
- i illuminator 
  - 0/1 -> off/on
  - H -> Translate Left X
  - L -> Translate Right X
  - J -> Translate Up Y
  - K -> Translate Down Y
  - A -> Translate into Z
  - D -> Translate out Z
